# Security Policy

## Supported Versions

Security fixes are provided for the latest published minor release.

## Reporting A Vulnerability

Do not open a public issue for a suspected vulnerability.

Use GitHub's private vulnerability reporting for this repository:

https://github.com/iifor/agent-creator/security/advisories/new

Include the affected package and version, reproduction steps, impact, and any
suggested mitigation. Maintainers will acknowledge the report as soon as
practical and coordinate disclosure after a fix is available.
